Quantum Materials Group Wiki
===================================

This **QMG** wiki is a key resource for all the theory, experimental, and data processing know-how
necessary for day to day activities in the group. The aim is to be able to browse and contribute 
knowledge that may not be fully detailed elsewhere.

.. note::

   This project is under active development.

Find the source code and contribute to this project at

https://github.com/quantum-materials-group/QMG-wiki.git
